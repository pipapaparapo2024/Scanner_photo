# pages/ - Страницы/экраны

Полноценные экраны, которые видят пользователи.

## Структура

- `HomePage/` - Главный экран (камера + история)
- `ScanHistoryPage/` - Список всех сканов
- `ScanDetailPage/` - Детали одного скана
- `MonetizationPage/` - Экран покупок и рекламы
- `SettingsPage/` - Настройки приложения

## Правила

- Каждая страница содержит `index.tsx` и опционально `ui/` для специфичных компонентов
- Импортирует из `shared`, `entities`, `features`, `widgets`
- Композирует виджеты и фичи в полноценные экраны
