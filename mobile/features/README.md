# features/ - Бизнес-фичи

Отдельные пользовательские сценарии: actions, UI, логика.

## Структура

- `camera-scan/` - Сканирование с камеры + OCR
  - `ui/` - Компоненты камеры и результата
  - `model/` - Хуки для логики сканирования
  - `api/` - Запросы к `/api/scans`
  - `lib/` - ML Kit интеграция
- `scan-history/` - Работа с историей сканов
  - `ui/` - Элементы списка
  - `model/` - Хуки для загрузки истории
- `monetization/` - Монетизация (покупки + реклама)
  - `ui/` - Кнопки покупки и рекламы
  - `model/` - Хуки для IAP и Ads
  - `api/` - Запросы к `/api/iap` и `/api/ads`
- `auth/` - Аутентификация
  - `ui/` - Экран входа
  - `model/` - Хуки для Firebase Auth

## Правила

- Каждая фича изолирована и может использоваться независимо
- Импортирует из `shared`, `entities`
- Содержит UI, бизнес-логику и API запросы
