# entities/ - Бизнес-сущности

Модели данных и их базовые операции.

## Структура

- `user/` - Пользователь и баланс
  - `model/types.ts` - Типы (`User`, `ScanCredits`)
  - `api/userApi.ts` - Запросы к `/api/users/me`
  - `model/useUser.ts` - Хук для работы с пользователем
- `scan/` - Модель скана
  - `model/types.ts` - Типы (`Scan`, `ScanDoc`)
  - `model/useScan.ts` - Базовые операции со сканом

## Правила

- Содержит только модели данных и базовые CRUD операции
- Импортирует только из `shared`
- Не содержит бизнес-логику (она в `features`)
