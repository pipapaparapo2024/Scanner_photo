# shared/ - Переиспользуемый код

Утилиты, константы, базовые компоненты без бизнес-логики.

## Структура

- `ui/` - Базовые UI компоненты (Button, Text, Card, Loading)
- `lib/` - Утилиты
  - `api/` - Базовый HTTP клиент (axios/fetch)
  - `storage/` - Работа с AsyncStorage
  - `format/` - Форматирование (даты, текст)
  - `native/` - Нативные модули (RuStore Pay, Яндекс Ads)
- `config/` - Конфигурация
  - `api.ts` - Базовый URL бэкенда
  - `firebase.ts` - Конфиг Firebase (если нужен на клиенте)
- `types/` - Общие типы
  - `api.ts` - Типы для API ответов

## Правила

- Не содержит бизнес-логику
- Максимально переиспользуемый код
- Не зависит от других слоев FSD
