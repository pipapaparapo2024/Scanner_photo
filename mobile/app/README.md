# app/ - Инициализация приложения

Слой инициализации приложения: точка входа, провайдеры, роутинг верхнего уровня.

## Структура

- `App.tsx` - Корневой компонент приложения
- `index.tsx` - Точка входа (регистрация нативного кода)
- `providers/` - Провайдеры (Firebase, Navigation, State)
- `router/` - Конфигурация навигации

## Правила

- Импортирует только из `shared`, `entities`, `features`, `widgets`, `pages`
- Не содержит бизнес-логики
- Только настройка и композиция
